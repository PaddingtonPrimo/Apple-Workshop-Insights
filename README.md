# Apple Workshop Insights

## Description
This repository contains the Apple Workshop Insights case study backed with robust insights focusing on workshops related to Apple products and services, showcasing user interaction, workshop statistics, and feedback.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/PaddingtonPrimo/Apple-Workshop-Insights.git
   cd Apple-Workshop-Insights
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## How to Run Locally
To run the project locally, execute:
```bash
npm run dev
```

## Build Instructions
To create a production build, run:
```bash
npm run build
```

## GitHub Pages Deployment
The deployed site can be found at: [https://PaddingtonPrimo.github.io/Apple-Workshop-Insights/](https://PaddingtonPrimo.github.io/Apple-Workshop-Insights/)  

## Features
- Animated statistics counters
- Interactive tabbed sections
- Timeline visualization
- Animated progress bars
- Premium dark theme

## Technologies Used
This project is built using React and Vite.
